## Description

This is my solution to the recruitment task.

## Requirements
```
node version >= 21.7.3
```
With this version of node, the project was tested and works correctly.

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```
